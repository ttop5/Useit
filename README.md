+ Echarts
+ Yarn
+ ES6
+ Less
+ ......

## Usage
```
git clone git@github.com:ttop5/useit.git
cd useit
yarn
yarn mock
yarn dev
```

View app: `127.0.0.1:8888` or view mock api: `127.0.0.1:3333` on your browser.
